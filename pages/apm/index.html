APM

2026-03-14

APM is the package manager for the Lua distribution ComEXE. It comes with a curated collection of Lua-only modules — all source code, no binaries — and is still a work in progress. It offers a subset of what's available on LuaRocks, focused exclusively on PUC Lua 5.5. That means modules for Awesome-WM, LuaJIT, OpenResty, Kong, TeX/LaTeX, Torch, and the like have been excluded.

The index is available, more modules are on the way...

Usage on Windows and Linux

lua55ce.exe -x --apm repo add https://pascalcombier.github.io/apm/packages/lua/5.5/index.zip
lua55ce.exe -x --apm list
lua55ce.exe -x --apm install uint-0.186
